# StreamingCockpit (STC)

StreamingCockpit is a DreamOS plugin for playback of IPTV live streams.

## Features
- Plays back IPTV streams. The first release focuses on PlutoTV streams.
- STC actually comprises 3 plugins: 
	- enigma2-plugin-extensions-streamingcockpit: GUI frontend in python2.
	- enigma2-plugin-ubuntu-streamingserver: Streaming server in python3
	- enigma2-plugin-systemplugins-ubuntucockpit: Ubuntu chroot subsystem providing a python3 environment.

## Limitations
- STC supports DreamOS only.
- STC is being tested on DM 920 and DM ONE only.
