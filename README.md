# StreamingCockpit (STC)

StreamingCockpit is the GUI frontend for the streamingserver plugin (which is running on the dreambox ubuntu subsystem)

![Screenshot](stc1.png)

## Features
- Plays back IPTV streams.

## Limitations
- STC supports DreamOS only.
- STC is being tested on DM 920 and DM ONE only.

## Languages
- english
- german
